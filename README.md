# Unitary matrix decomposition with complex Givens rotations

Этот проект — тестовое задание, попытка реализации разложения комплексной унитарной матрицы на последовательность комплексных Givens-вращений и диагональную фазовую матрицу.

Целевая форма разложения:

$$
W \approx \left(\prod_k G_{i_k j_k}(\theta_k, \phi_k)\right) D
$$

где $$G_{ij}$$ — комплексные Givens-блоки, а $$D$$ — диагональная матрица фаз.[1][2]

## Статус проекта

На текущем этапе проект **частично работает**:

- Генерация тестовых унитарных матриц из `scipy.stats.unitary_group.rvs`, что даёт корректные элементы группы $$U(N)$$.
- Можно проверить что локальные шаги зануления целеового элемента с помощью комплексных Givens-вращений дают машинный 0;
- Последовательное применение Givens-вращений к элементам под диагональю успешно приводит первый столбец к виду `(*, 0, 0, 0)^T` с остатками порядка `1e-16`, что подтверждает корректность локальной схемы зануления и порядка обхода внутри столбца.
- восстановленная матрица остаётся унитарной;

## Что пока не работает

- процедура декомпозиции корректно выполняет последовательное зануление поддиагональных элементов;
- итоговая реконструкция `Wmesh` остаётся унитарной;
- однако норма ошибки восстановления `||W - Wmesh||` остаётся порядка `2.4-3.0` вместо значений, близких к машинной точности.

Текущая реализация пока не воспроизводит точную факторизацию исходной матрицы в виде произведения Givens-вращений и диагональной фазовой матрицы.

## Следующие шаги

- Сравнить порядок вращений и фазовые соглашения с `qml.math.decomposition.givens_decomposition`.
- Сравнить текущую схему с `openfermion.linalg.givens_decomposition` или `givens_decomposition_square`.
- Провести отдельную аналитическую и численную проверку для матриц размеров `2x2` и `3x3`.
